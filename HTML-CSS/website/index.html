历史

温格时代

自格拉汉姆离开之后，阿森纳请来了一位默默无闻的法国主帅。没有人料到，他将给俱乐部带来如此深远的改变。

阿森纳球场：海布里

阿森纳球场，它的另一个名字“海布里球场”更为人们所熟知。自1913年至2006年期间，它一直是俱乐部的主场。

阿森纳的传承

阿森纳不仅仅是一家历史悠久的俱乐部。他是历史、传统和胜利的代名词。除了满载荣誉的过去和现在，阿森纳同样关注自身的发展与未来。
